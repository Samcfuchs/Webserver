About this site

If you have not been invited to this site, please leave so that there is less traffic for legitimate users. Thank you.

This site is the host for all of my major projects and programs. It's now accessible to anyone with its IP address, which could create problems with traffic, if it got out of control. It's now running on a Raspberry Pi 2 Model B! This is running the most recent release of Raspbian, which handles it relatively well, but probably couldn't deal with much traffic. For this reason, it is crucial that this IP remains private. I've added CSS and Javascript, but there's not much there yet.
